About /dev/rs0

Low-level systems, cryptography, hardware hacking, self-hosted AI, and the occasional weird idea that just might work.

I'm the kind of person who builds from scratch before reaching for a library — not out of stubbornness, but because the learning happens in the implementation. I'm drawn to hard problems with challenging constraints, the kind where understanding one layer reveals something unexpected about the layer below. I'd rather read the spec than the tutorial. Some projects ship, some don't — abandoned code is just tuition I've already paid. If there's an idea that probably won't work but might, I'd rather find out than wonder.

Built with Noise, a minimal static site generator.
