# ryonsherman.github.io

Personal site hosted via [GitHub Pages](https://pages.github.com/).

## Contents

- **Blog** — generated by [Noise](https://github.com/ryonsherman/noise), a minimal static site generator. Posts are markdown files with front matter under `posts/`.
- **Canary** — a Gradle/Kotlin proof-of-life experiment, built to `canary/`.

## Development

Source lives in the `noise` repo. See its README for build instructions.
